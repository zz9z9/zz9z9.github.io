Hash Join

н•ҙмӢң н…Ңмқҙлё”мқ„ мӮ¬мҡ©н•ҙм„ң л§Өм№ӯлҗҳлҠ” н–үмқ„ м°ҫлҠ” мЎ°мқё л°©мӢқ

  • Hash JoinмқҖ нҒ¬кІҢ build phaseмҷҖ probe phase л‘җ лӢЁкі„лЎң кө¬м„ұлҗңлӢӨ.

1. Build Phase

SELECT given_name, country_name
FROM persons
JOIN countries ON persons.country_id = countries.country_id;
  • н•ңмӘҪ мһ…л Ҙмқ„ кё°л°ҳмңјлЎң in-memory н•ҙмӢң н…Ңмқҙлё”мқ„ л§Ңл“ лӢӨ.
  • мқҙл•Ң н•ҙмӢң н…Ңмқҙлё”мқҳ keyлҠ” мЎ°мқё мЎ°кұҙ(м—¬кё°м„ңлҠ” country_id)мқҙлӢӨ.
  • ліҙнҶө лҚ” мһ‘мқҖ мһ…л Ҙ(л°”мқҙнҠё нҒ¬кё° кё°мӨҖ)мқ„ н•ҙмӢң н…Ңмқҙлё”лЎң л§Ңл“ лӢӨ.
    • л©”лӘЁлҰ¬м—җ мҳ¬л Өм•ј н•ҳкё° л•Ңл¬ём—җ мһ‘мқҖ мӘҪмқҙ мң лҰ¬н•Ё.
  • мҳҲмӢңм—җм„ң countries н…Ңмқҙлё”мқ„ build inputмңјлЎң м„ нғқн–ҲлӢӨкі  к°Җм •н•ҳл©ҙ:
    • countriesмқҳ лӘЁл“  country_idлҘј нӮӨлЎң н•ҙм„ң н•ҙмӢң н…Ңмқҙлё”м—җ м ҖмһҘ
    • мқҙ кіјм •мқ„ л§Ҳм№ҳл©ҙ build phase мў…лЈҢ

(м¶ңмІҳ : https://dev.mysql.com/blog-archive/hash-join-in-mysql-8/)

2. Probe Phase

  • мқҙм ң м„ңлІ„лҠ” probe input(мҳҲм ңм—җм„ңлҠ” persons)мқ„ мқҪкё° мӢңмһ‘н•ңлӢӨ.
  • к°Ғ rowм—җ лҢҖн•ҙ:
    • persons.country_id к°’мқ„ нӮӨлЎң н•ҙм„ң, н•ҙмӢң н…Ңмқҙлё”м—җм„ң л§Өм№ӯлҗҳлҠ” rowлҘј м°ҫлҠ”лӢӨ.
    • л§Өм№ӯлҗҳлҠ” rowк°Җ мһҲмңјл©ҙ, к·ё л‘җ rowлҘј н•©міҗм„ң(joined row) нҒҙлқјмқҙм–ёнҠёлЎң ліҙлӮёлӢӨ.

  • мҰү, personsмқҳ лӘЁл“  rowлҘј н•ң лІҲм”©л§Ң мқҪкі , н•ҙмӢң н…Ңмқҙлё”м—җм„ң мғҒмҲҳ мӢңк°„(constant time)мңјлЎң л§Өм№ӯмқ„ м°ҫкІҢ лҗңлӢӨ.

  • мқҙ л°©мӢқмқҙ мһҳ лҸҷмһ‘н•ҳл Өл©ҙ build input м „мІҙлҘј л©”лӘЁлҰ¬м—җ лӢҙмқ„ мҲҳ мһҲм–ҙм•ј н•ңлӢӨ.
  • MySQLм—җм„ңлҠ” мӮ¬мҡ© к°ҖлҠҘн•ң л©”лӘЁлҰ¬ нҒ¬кё°лҘј join_buffer_size мӢңмҠӨн…ң ліҖмҲҳк°Җ м ңм–ҙн•ңлӢӨ.
    • мқҙ к°’мқҖ мӢӨн–ү мӨ‘(runtime)м—җлҸ„ мЎ°м •н•  мҲҳ мһҲлӢӨ.

(м¶ңмІҳ : https://dev.mysql.com/blog-archive/hash-join-in-mysql-8/)

Spill to disk

build inputмқ„ л©”лӘЁлҰ¬м—җ лӢӨ мҳ¬лҰҙ мҲҳ м—ҶлҠ” кІҪмҡ°, м—¬лҹ¬ к°ңмқҳ нҢҢмқјлЎң лӮҳлҲ м„ң л””мҠӨнҒ¬м—җ м ҖмһҘ

1. Probe Phaseм—җм„ңмқҳ Spill мІҳлҰ¬

  • MySQLмқҖ build inputмқ„ м—¬лҹ¬ к°ңмқҳ chunk нҢҢмқјлЎң лӮҳлҲ„м–ҙ м ҖмһҘ:
    • мҳҲлҘј л“Өм–ҙ build inputмқҙ 1GBмқёлҚ° л©”лӘЁлҰ¬лҠ” 256MBлқјл©ҙ
    • лҢҖлһө 4к°ңмқҳ chunk нҢҢмқјлЎң лӮҳлҲ м„ң л””мҠӨнҒ¬м—җ м“°лҠ” мӢқ
    • лӢЁ, MySQLмқҖ лӮҙл¶Җм ҒмңјлЎң chunk нҢҢмқј мөңлҢҖ 128к°ңк№Ңм§Җл§Ң л§Ңл“Ө мҲҳ мһҲмқҢ
  • м–ҙл–Ө rowк°Җ м–ҙл–Ө chunk нҢҢмқјлЎң л“Өм–ҙк°Ҳм§ҖлҠ” мЎ°мқё нӮӨ(мҳҲ: country_id)м—җ лҢҖн•ҙ н•ҙмӢң к°’мқ„ кі„мӮ°н•ҙм„ң м •н•ңлӢӨ.
    • мҰү, к°ҷмқҖ нӮӨ к°’мқ„ к°Җм§„ rowл“ӨмқҖ н•ӯмғҒ к°ҷмқҖ chunk нҢҢмқјм—җ л“Өм–ҙк°ҖкІҢлҗңлӢӨ.
  • мқҙл•Ң мӮ¬мҡ©н•ҳлҠ” н•ҙмӢң н•ЁмҲҳлҠ”, лӮҳмӨ‘м—җ л©”лӘЁлҰ¬ м•Ҳм—җм„ң мӮ¬мҡ©н•ҳлҠ” н•ҙмӢң н•ЁмҲҳмҷҖлҠ” лӢӨлҘҙлӢӨ.

2. Probe Phaseм—җм„ңмқҳ Spill мІҳлҰ¬

  • build input н•ҙмӢң н…Ңмқҙлё”м—җ мқјм№ҳн•ҳлҠ” rowк°Җ мһҲлҠ”м§Җ м°ҫм•„м•јн•ҳлҠ”лҚ°, build inputмқҳ мқјл¶Җк°Җ л””мҠӨнҒ¬м—җ мһҲкё° л•Ңл¬ём—җ probe inputмқҳ rowлҸ„ л””мҠӨнҒ¬м—җ мһҲлҠ” chunkмҷҖ м§қмқ„ л§һм¶°м•ј н•ңлӢӨ.
  • л”°лқјм„ң, probe inputлҸ„ build inputмқ„ chunk fileм—җ л¶„л°°н• л•Ң мӮ¬мҡ©н•ң лҸҷмқјн•ң н•ҙмӢң н•ЁмҲҳлЎң кі„мӮ°н•ҙм„ң chunk нҢҢмқјмқ„ л§Ңл“ лӢӨ.
    • мқҙл ҮкІҢ н•ҳл©ҙ мЎ°мқё нӮӨк°Җ к°ҷмқҖ rowл“ӨмқҖ л°ҳл“ңмӢң к°ҷмқҖ chunk мҢҚ(= build chunk + probe chunk)м—җ л“Өм–ҙк°ҖкІҢ лҗңлӢӨ.

3. Chunk File мІҳлҰ¬

  • лӘЁл“  chunk нҢҢмқј мҢҚмқ„ мІҳлҰ¬:
    • мІ« лІҲм§ё build chunkлҘј мқҪм–ҙм„ң л©”лӘЁлҰ¬м—җ н•ҙмӢң н…Ңмқҙлё”лЎң л§Ңл“ лӢӨ.
    • к·ём—җ лҢҖмқ‘н•ҳлҠ” probe chunkлҘј мқҪм–ҙм„ң н•ҙмӢң н…Ңмқҙлё”кіј л§Өм№ӯмӢңнӮЁлӢӨ.
    • лҒқлӮҳл©ҙ л‘җ лІҲм§ё chunk мҢҚмңјлЎң л„ҳм–ҙк°„лӢӨ.
    • мқҙлҹ° мӢқмңјлЎң лӘЁл“  chunk нҢҢмқј мҢҚмқ„ мҲңм°Ём ҒмңјлЎң мІҳлҰ¬н•ңлӢӨ.
  • build chunkлҘј мқҪм–ҙм„ң л©”лӘЁлҰ¬м—җ н•ҙмӢң н…Ңмқҙлё”лЎң л§Ңл“Өл•Ңмқҳ н•ҙмӢң н•ЁмҲҳлҠ” inputмқ„ chunk fileлЎң л¶„л°°н•  л•Ңмқҳ н•ҙмӢң н•ЁмҲҳмҷҖлҠ” лӢӨлҘҙлӢӨ.
    • к°ҷмқҖ н•ҙмӢң н•ЁмҲҳлҘј м“°л©ҙ, chunk нҢҢмқјм—җ мһҲлҠ” rowл“Өмқҙ м „л¶Җ к°ҷмқҖ н•ҙмӢң к°’м—җ лӘ°л Өм„ң, н•ҙмӢң н…Ңмқҙлё”мқҙ н•ңмӘҪмңјлЎң мҸ лҰ¬лҠ” л¬ём ң(hash collision)к°Җ мғқкёҙлӢӨ.

(м¶ңмІҳ : https://dev.mysql.com/blog-archive/hash-join-in-mysql-8/)

  • лӘЁл“  chunk нҢҢмқј мҢҚмқҙ мЎ°мқёмқ„ л§Ҳм№ҳл©ҙ лӢӨмқҢмқ„ мҲҳн–ү:

    • нҢҢмқј н•ёл“Өмқ„ лӢ«мқҖ лӢӨмқҢ, OS л ҲлІЁм—җм„ң unlink() нҳём¶ңмқ„ мҲҳн–ү
