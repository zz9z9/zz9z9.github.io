Infra - Tomcat & Apache HTTP Server м—°лҸҷ

Lee JaeYoon Feb 10, 2022 2022-02-10T13:25:00+09:00
12 min

л“Өм–ҙк°Җкё° м „

Tomcatкіј Apache HTTP Server м—°лҸҷкіј кҙҖл Ёлҗң кё°мҙҲ м§ҖмӢқмқ„ мӮҙнҺҙліҙкі , лӢӨмқҢ нҢҢнҠём—җм„ңлҠ” мӢӨм ңлЎң м—°лҸҷн•ҳлҠ” мһ‘м—…мқ„ м§„н–үн•ҙліҙл Өкі  н•ңлӢӨ.

Apache Tomcat Connectors

Apache Tomcat Connectors н”„лЎңм қнҠёлҠ” Tomcat н”„лЎңм қнҠёмқҳ мқјл¶Җмқҙл©°, мӣ№ м„ңлІ„лҘј Tomcat л°Ҹ кё°нғҖ л°ұм—”л“ңмҷҖ м—°кІ°н•ҳкё° мң„н•ң мӣ№ м„ңлІ„ н”Ңлҹ¬к·ёмқёмқ„ м ңкіөн•ңлӢӨ.

  • м§ҖмӣҗлҗҳлҠ” мӣ№ м„ңлІ„лҠ” лӢӨмқҢкіј к°ҷлӢӨ.
мӣ№ м„ңлІ„н”Ңлҹ¬к·ёмқё
Apache HTTP Servermod_jk
Microsoft IISISAPI redirector

  • н”Ңлҹ¬к·ёмқё лҢҖл¶Җл¶„мқҳ кё°лҠҘмқҖ лӘЁл“  мӣ№ м„ңлІ„м—җм„ң лҸҷмқјн•ҳл©°, м„ёл¶Җ мӮ¬н•ӯмқҖ мӣ№ м„ңлІ„лі„лЎң лӢӨлҘј мҲҳ мһҲлӢӨ.
  • лӘЁл“  кІҪмҡ°м—җ н”Ңлҹ¬к·ёмқёмқҖ Apache JServ Protocol(AJP) лқјлҠ” нҠ№мҲҳ н”„лЎңнҶ мҪңмқ„ мӮ¬мҡ©н•ҳм—¬ л°ұм—”л“ңм—җ м—°кІ°н•ңлӢӨ.
  • AJPлҘј м§Җмӣҗн•ҳлҠ” кІғмңјлЎң м•Ңл Өм§„ л°ұм—”л“ңлҠ” Apache Tomcat, Jetty, JBoss л“ұмқҙ мһҲлӢӨ.
  • н”„лЎңнҶ мҪңм—җлҠ” ajp12, ajp13, ajp14мқҳ 3к°Җм§Җ лІ„м „мқҙ мһҲм§Җл§Ң, м•„м§Ғк№Ңм§Җ лҢҖл¶Җл¶„мқҳ кІҪмҡ° ajp13л§Ң мӮ¬мҡ©н•ңлӢӨ.
    • AJP 1.3 лҳҗлҠ” AJPv13мқҙлқјкі  л¶ҖлҘҙм§Җл§Ң мЈјлЎң ajp13лқјлҠ” мқҙлҰ„мқ„ мӮ¬мҡ©н•ңлӢӨ.
    • нҶ°мәЈмқҖ 3.2 лІ„м „ мқҙнӣ„лЎң ajp13мқ„ м§Җмӣҗн•ңлӢӨ.

Apache HTTP Server - Tomcat м—°лҸҷ

мң„м—җм„ң мӮҙнҺҙлҙӨл“Ҝмқҙ mod_jkлҘј мӮ¬мҡ©н•ҳм—¬ м—°лҸҷн•ңлӢӨ. (мөңмҶҢ лІ„м „ : Apache 1.3, Tomcat 3.2)

  • мӣ№ м„ңлІ„(Apache HTTP Server)лҠ” нҒҙлқјмқҙм–ёнҠё HTTP мҡ”мІӯмқ„ кё°лӢӨлҰ¬кі  мһҲлҠ”лӢӨ.
    • мҡ”мІӯмқҙ лҸ„м°©н•ҳл©ҙ м„ңлІ„лҠ” н•„мҡ”н•ң мҪҳн…җмё лҘј м ңкіөн•ҳм—¬ мҡ”мІӯмқ„ мІҳлҰ¬н•ҳлҠ” лҚ° н•„мҡ”н•ң лӘЁл“  мһ‘м—…мқ„ мҲҳн–үн•ңлӢӨ.
  • м„ңлё”лҰҝ м»Ён…Ңмқҙл„Ҳ(мң„мқҳ кІҪмҡ° Tomcat)лҘј м¶”к°Җн•ҳл©ҙ мӣ№ м„ңлІ„лҠ” лӢӨмқҢкіј к°ҷмқҙ лҸҷмһ‘н•ңлӢӨ.
    • мҡ”мІӯмқ„ мІҳлҰ¬н•ҳкё° м „м—җ, м„ңлё”лҰҝ м»Ён…Ңмқҙл„Ҳ м–ҙлҢ‘н„° лқјмқҙлёҢлҹ¬лҰ¬лҘј лЎңл“ңн•ҳкі  мҙҲкё°нҷ”н•ңлӢӨ.
    • мҡ”мІӯмқҙ мҳӨл©ҙ, нҠ№м • мҡ”мІӯмқҙ м„ңлё”лҰҝм—җ мҶҚн•ҳлҠ”м§Җ нҷ•мқён•ҙм•ј н•ҳл©°, м„ңлё”лҰҝм—җ мҶҚн•ҳлҠ” кІҪмҡ° м–ҙлҢ‘н„°к°Җ мҡ”мІӯмқ„ л°ӣм•„ мІҳлҰ¬н•ҳлҸ„лЎқ н•ңлӢӨ.
    • м–ҙлҢ‘н„°лҠ” мқјл°ҳм ҒмңјлЎң мҡ”мІӯ URLмқҳ мқјл¶Җ нҢЁн„ҙкіј мҡ”мІӯмқ„ ліҙлӮј мң„м№ҳлҘј кё°л°ҳмңјлЎң м–ҙл–Ө мҡ”мІӯмқ„ мІҳлҰ¬н• м§Җ м•Ңм•„м•јн•ңлӢӨ.

mod_jk кҙҖл Ё мӮ¬н•ӯ

mod_jkлҠ” л‘җ к°ңмқҳ м—”нӢ°нӢ°к°Җ н•„мҡ”н•ҳлӢӨ.

  1. mod_jk.xxx
    • Apache HTTP Server лӘЁл“Ҳ
      • мҡҙмҳҒ мІҙм ңм—җ л”°лқј mod_jk.so, mod_jk.nlm лҳҗлҠ” MOD_JK.SRVPGM
    • лЁјм Җ Apache HTTP Serverмқҳ лӘЁл“Ҳ л””л үнҶ лҰ¬(мҳҲ:/usr/lib/apache)м—җ м„Өм№ҳлҗҳм–ҙм•ј н•ҳл©° httpd.conf нҢҢмқјм—җм„ң лЎңл“ңлҗҳм–ҙм•ј н•ңлӢӨ.
      • ex : LoadModule jk_module modules/mod_jk.so
  2. workers.properties
    • Tomcat н”„лЎңм„ёмҠӨк°Җ мӮ¬мҡ©н•ҳлҠ” нҳёмҠӨнҠё, нҸ¬нҠё л“ұмқ„ м •мқҳн•ҳлҠ” нҢҢмқјмқҙлӢӨ. (м•„лһҳ Worker нҢҢнҠё м°ёмЎ°)
    • workers.propertiesнҢҢмқјмқҖ ${APACHE_HOME}/conf л””л үнҶ лҰ¬м—җ мһҲлӢӨ.
    • httpd.confм—җм„ң н•ҙлӢ№ нҢҢмқјмқҳ кІҪлЎңлҘј м •мқҳн•ҙм•ј н•ңлӢӨ.
      • ex : JkWorkersFile /etc/httpd/conf/workers.properties

м—°лҸҷмӢң мЈјмқҳмӮ¬н•ӯ

  • ApacheмҷҖ Tomcatмқҙ лҸҷмқјн•ң нҢҢмқј мӢңмҠӨн…ң мң„м№ҳм—җм„ң мҪҳн…җмё лҘј м ңкіөн•ҳлҸ„лЎқ кө¬м„ұлҗң кІҪмҡ°, Apacheк°Җ WEB-INF л””л үнҶ лҰ¬ лҳҗлҠ” JSP мҶҢмҠӨ мҪ”л“ңмҷҖ к°ҷмқҖ мҪҳн…җмё (м„ңлё”лҰҝ м»Ён…Ңмқҙл„Ҳк°Җ мІҳлҰ¬н•ҙм•јн•ҳлҠ” мҪҳн…җмё )лҘј м ңкіөн•ҳм§Җ м•ҠлҸ„лЎқ н•ҙм•јн•ңлӢӨ.
    • Apache DocumentRootк°Җ Tomcat Hostмқҳ appBase лҳҗлҠ” Contextмқҳ docBaseмҷҖ кІ№м№ҳл©ҙ л°ңмғқн•  мҲҳ мһҲлӢӨ.
    • Apache Alias м§ҖмӢңл¬ёмқ„ Tomcat Hostмқҳ appBase лҳҗлҠ” Contextмқҳ docBaseмҷҖ н•Ёк»ҳ мӮ¬мҡ©н•ҳл©ҙ л°ңмғқн•  мҲҳ мһҲлӢӨ.

Worker

Tomcat Worker

Tomcat WorkerлҠ” мқјл¶Җ мӣ№ м„ңлІ„лҘј лҢҖмӢ н•ҳм—¬ м„ңлё”лҰҝ лҳҗлҠ” кё°нғҖ мҪҳн…җмё лҘј мӢӨн–үн•ҳкё° мң„н•ҙ лҢҖкё° мӨ‘мқё Tomcat мқёмҠӨн„ҙмҠӨмқҙлӢӨ.

вҖ» м°ёкі  : кіөмӢқ л¬ём„ңлҘј ліҙл©ҙ Tomcat WorkerлҘј лӢӨмқҢкіј к°ҷмқҙ м •мқҳн•ңлӢӨ.

  • вҖңA Tomcat worker is a Tomcat instance that is waiting to execute servlets on behalf of some web server. For example, we can have a web server such as the Apache HTTP Server forwarding servlet requests to a Tomcat process (the worker) running behind it.вҖқ

  • лҢҖн‘ңм ҒмңјлЎң лӢӨмқҢкіј к°ҷмқҖ кІҪмҡ°, м—¬лҹ¬ к°ңмқҳ workerлҘј кө¬м„ұн•  мҲҳлҸ„ мһҲлӢӨ.

    • лӘЁл“  к°ңл°ңмһҗк°Җ лҸҷмқјн•ң мӣ№ м„ңлІ„лҘј кіөмң н•ҳм§Җл§Ң, мһҗмӢ мқҳ workerлҘј мҶҢмң н•ҳлҠ” к°ңл°ң нҷҳкІҪмқҙ н•„мҡ”н•ң кІҪмҡ°.
    • н•ҳлӮҳмқҳ мӣ№ м„ңлІ„м—җм„ң м—¬лҹ¬ мӮ¬мқҙнҠёлҘј м ңкіөн•ҳкё° мң„н•ҙ, нҶ°мәЈ н”„лЎңм„ёмҠӨлі„ к°ҖмғҒ нҳёмҠӨнҠёк°Җ н•„мҡ”н•ң кІҪмҡ°.
    • лЎңл“ң л°ёлҹ°мӢұмқ„ м ңкіөн•ҳкі мһҗ н•ҳлҠ” кІҪмҡ°.
      • мҰү, мһҗмІҙ лЁёмӢ м—җм„ң м—¬лҹ¬ workerлҘј мӢӨн–үн•ҳкі  мҡ”мІӯмқ„ л¶„л°°н•ҙм•ј н•ңлӢӨ.

    • мқёмҠӨн„ҙмҠӨнҷ”лҗң workerлҠ” мһҗмӢ м—җкІҢ л§өн•‘лҗң нҠ№м • мҡ”мІӯмқ„ м „лӢ¬л°ӣмқ„ мҲҳ мһҲлӢӨ.
  • worker.listлҠ” м—¬лҹ¬ лІҲ мӮ¬мҡ©н•  мҲҳ мһҲлӢӨ.

lb Worker мӮҙнҺҙліҙкё°

лЎңл“ң л°ёлҹ°мӢұ workerлҠ” мӢӨм ңлЎң Tomcat workerмҷҖ нҶөмӢ н•ҳм§Җ м•Ҡкі , мӣ№ м„ңлІ„ н”Ңлҹ¬к·ёмқё worker кҙҖлҰ¬лҘј лӢҙлӢ№н•ңлӢӨ.

  • кҙҖлҰ¬н•ҳлҠ” л¶Җл¶„мқҖ лӢӨмқҢкіј к°ҷлӢӨ.
    • мӣ№ м„ңлІ„ н”Ңлҹ¬к·ёмқё workerлҘј мқёмҠӨн„ҙмҠӨнҷ”
    • кҙҖлҰ¬лҗҳлҠ” workerмқҳ лЎңл“ң л°ёлҹ°мӢұ factor(lbfactor)лҘј мӮ¬мҡ©н•ҳм—¬ к°ҖмӨ‘м№ҳ лқјмҡҙл“ң лЎңл№Ҳ(weighted round-robin) лЎңл“ң л°ёлҹ°мӢұмқ„ мҲҳн–ү
      • к°ҖмӨ‘м№ҳ лқјмҡҙл“ң лЎңл№Ҳм—җм„ңлҠ” lbfactorк°Җ лҶ’мқ„мҲҳлЎқ лҚ” к°•л Ҙн•ң мӢңмҠӨн…ң(лҚ” л§ҺмқҖ мҡ”мІӯмқ„ мІҳлҰ¬н•Ё)мқ„ мқҳлҜё
    • лҸҷмқјн•ң Tomcat workerм—җм„ң мӢӨн–үлҗҳлҠ” лҸҷмқјн•ң м„ём…ҳм—җ мҶҚн•ҳлҠ” мҡ”мІӯ мң м§Җ(м„ём…ҳ кі м •)
    • мӢӨнҢЁн•ң Tomcat workerлҘј мӢқлі„н•ҳм—¬ мҡ”мІӯмқ„ мқјмӢң мӨ‘м§Җн•ҳкі , lb workerк°Җ кҙҖлҰ¬н•ҳлҠ” лӢӨлҘё workerлЎң fail-over

  • кІ°кіјм ҒмңјлЎң, lb workerк°Җ кҙҖлҰ¬н•ҳлҠ” workerл“ӨмқҖ лЎңл“ң л°ёлҹ°мӢұлҗҳкі (lbfactor л°Ҹ нҳ„мһ¬ мӮ¬мҡ©мһҗ м„ём…ҳмқ„ кё°л°ҳмңјлЎң) мһҘм•  мЎ°м№ҳк°Җ мқҙлЈЁм–ҙм§ҖлҜҖлЎң лӢЁмқј Tomcat н”„лЎңм„ёмҠӨ мў…лЈҢлЎң мқён•ҙ м „мІҙ мӮ¬мқҙнҠёк°Җ л§Ҳл№„лҗҳм§Җ м•ҠлҠ”лӢӨ.

  • лӢӨмқҢмқҖ lb workerк°Җ к°Җм§Ҳ мҲҳ мһҲлҠ” мҶҚм„ұмқҙлӢӨ.
