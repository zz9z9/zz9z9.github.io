WEB - Content-Security-Policy(CSP) мӮҙнҺҙліҙкё°

Lee JaeYoon Mar 15 2025-03-15T22:25:00+09:00
Apr 4 2025-04-04T00:47:55+09:00 11 min

Content-Security-Policy ?

мҪҳн…җмё  ліҙм•Ҳ м •мұ…(CSP)мқҖ нҠ№м • мң нҳ•мқҳ ліҙм•Ҳ мң„нҳ‘мқҳ мң„н—ҳмқ„ л°©м§Җн•ҳкұ°лӮҳ мөңмҶҢнҷ”н•ҳлҠ” лҚ° лҸ„мӣҖмқҙ лҗҳлҠ” кё°лҠҘмңјлЎң
мӣ№мӮ¬мқҙнҠём—җм„ң лёҢлқјмҡ°м ҖлЎң ліҙлӮҙлҠ” мқјл Ёмқҳ м§Җм№ЁмңјлЎң кө¬м„ұлҗҳм–ҙ мһҲмңјл©°, лёҢлқјмҡ°м Җм—җм„ң мҪ”л“ңк°Җ мҲҳн–үн•  мҲҳ мһҲлҠ” мһ‘м—…м—җ м ңн•ңмқ„ л‘җлҸ„лЎқ м§ҖмӢң

CSP м „лӢ¬ л°©мӢқ

  • HTTP мқ‘лӢө н—ӨлҚ”
    • л©”мқё л¬ём„ңлҝҗл§Ң м•„лӢҲлқј лӘЁл“  мҡ”мІӯм—җ лҢҖн•ң мқ‘лӢөм—җ м Ғмҡ©лҗҳм–ҙм•ј н•ңлӢӨ.
  • <meta> нғңк·ёмқҳ http-equiv мҶҚм„ұмңјлЎң м •мқҳ

м •мұ…

  • м •мұ…мқҖ м„ёлҜёмҪңлЎ мңјлЎң кө¬л¶„лҗң мқјл Ёмқҳ м§ҖмӢңл¬ё(Fetch directives)мңјлЎң кө¬м„ұ
  • к°Ғ м§ҖмӢңл¬ёмқҖ ліҙм•Ҳ м •мұ…мқҳ лӢӨлҘё мёЎл©ҙмқ„ м ңм–ҙ

мҳҲмӢң

Content-Security-Policy: default-src 'self'; img-src 'self' example.com

  • м§ҖмӢңл¬ё1 : default-src 'self'
    • лӢӨлҘё лҰ¬мҶҢмҠӨ мң нҳ•м—җ лҢҖн•ҙ лӢӨлҘё м •мұ…мқ„ м„Өм •н•ҳлҠ” кө¬мІҙм Ғмқё м§ҖмӢңм–ҙк°Җ м—ҶлҠ” н•ң, л¬ём„ңмҷҖ лҸҷмқјн•ң м¶ңмІҳмқҳ лҰ¬мҶҢмҠӨл§Ң лЎңл“ңн•ҳлқјкі  лёҢлқјмҡ°м Җм—җ м§ҖмӢң
  • м§ҖмӢңл¬ё2 : img-src 'self' example.com
    • лёҢлқјмҡ°м Җм—җ лҸҷмқјн•ң м¶ңмІҳмқҳ мқҙлҜём§ҖлӮҳ example.comм—җм„ң м ңкіөлҗң мқҙлҜём§ҖлҘј лЎңл“ңн•ҳлқјкі  м§ҖмӢң

Fetch directives

лҰ¬мҶҢмҠӨ(JavaScript, CSS, мқҙлҜём§Җ л“ұ)м—җ лҢҖн•ң н—Ҳмҡ©лҗң лІ”мЈј(лҸ„л©”мқё, URL, н”„лЎңнҶ мҪң, лҳҗлҠ” нҢЁн„ҙ л“ұ)лҘј м§Җм •
мҳҲлҘј л“Өм–ҙ, script-srcлҠ” JavaScript лЎңл“ңлҘј н—Ҳмҡ©н•ҳлҠ” лІ”мЈјлҘј м„Өм •
м „мІҙ м§ҖмӢңм–ҙ лӘ©лЎқмқҖ м—¬кё°м—җм„ң нҷ•мқён•  мҲҳ мһҲлӢӨ.

  • default-src : м§ҖмӢңл¬ёмқҙ лӘ…мӢңм ҒмңјлЎң лӮҳм—ҙлҗҳм§Җ м•ҠмқҖ лӘЁл“  лҰ¬мҶҢмҠӨм—җ лҢҖн•ң лҢҖмІҙ м •мұ…мқ„ м„Өм •

  • к°Ғ Fetch directivesлҠ” лӢЁмқј нӮӨмӣҢл“ң 'none' лҳҗлҠ” кіөл°ұмңјлЎң кө¬л¶„лҗң н•ҳлӮҳ мқҙмғҒмқҳ м¶ңмІҳ н‘ңнҳ„мӢқ(source expressions)мңјлЎң м§Җм •

    • none : н•ҙлӢ№ лҰ¬мҶҢмҠӨ лЎңл“ң мһҗмІҙлҘј м°ЁлӢЁ
    • self : л¬ём„ң мһҗмІҙмҷҖ лҸҷмқјн•ң м¶ңмІҳ(origin)мқҳ лҰ¬мҶҢмҠӨл§Ң н—Ҳмҡ©
    • л‘җ к°ң мқҙмғҒмқҳ м¶ңмІҳ н‘ңнҳ„мӢқмқҙ лӮҳм—ҙлҗң кІҪмҡ° н•ҳлӮҳлқјлҸ„ лҰ¬мҶҢмҠӨлҘј н—Ҳмҡ©н•ҳл©ҙ лҰ¬мҶҢмҠӨк°Җ н—Ҳмҡ©лҗЁ
      • н•ҳм§Җл§Ң, мҳҲмҷём ҒмңјлЎң noneмқҙ к°ҷмқҙ мһҲмңјл©ҙ лӢӨлҘё м¶ңмІҳ н‘ңнҳ„мӢқл“ӨмқҖ л¬ҙмӢңлҗЁ

мҳҲмӢң

Content-Security-Policy: default-src 'self'; img-src 'self' example.com

  • default-srcлҠ” лӢЁмқј м¶ңмІҳ н‘ңнҳ„мӢқ selfлЎң кө¬м„ұ.
  • img-srcлҠ” л‘җ к°ңмқҳ м¶ңмІҳ н‘ңнҳ„мӢқ selfмҷҖ example.comмңјлЎң кө¬м„ұ

вҖ» Fetch ?

  • fetchлҠ” мқјл°ҳм ҒмңјлЎң мҳҒм–ҙм—җм„ң вҖңк°Җм ёмҳӨлӢӨвҖқ лҳҗлҠ” вҖңл¶Ҳлҹ¬мҳӨлӢӨвҖқлқјлҠ” мқҳлҜё
  • fetchлҠ” лёҢлқјмҡ°м Җк°Җ мҷёл¶Җ лҰ¬мҶҢмҠӨлҘј мҡ”мІӯн•ҳкұ°лӮҳ к°Җм ёмҳӨлҠ” н–үмң„
  • мҰү, Fetch directivesлһҖ мӣ№ л¬ём„ңк°Җ л¶Ҳлҹ¬мҳ¬ мҲҳ мһҲлҠ” лҰ¬мҶҢмҠӨмқҳ мў…лҘҳлӮҳ м¶ңмІҳлҘј м ңн•ңн•ҳлҠ” м§ҖмӢңм–ҙ

м¶ңмІҳ н‘ңнҳ„мӢқ нҳ•мӢқ мӮҙнҺҙліҙкё°

<host-source> л°Ҹ <scheme-source> нҳ•мӢқмқҖ л”°мҳҙн‘ңлЎң л¬¶м–ҙм„ңлҠ” м•Ҳлҗҳл©°, лӢӨлҘё лӘЁл“  нҳ•мӢқмқҖ мһ‘мқҖ л”°мҳҙн‘ңлЎң л¬¶м–ҙм•ј н•ңлӢӨ.

<host-source>

н•ҙлӢ№ лҰ¬мҶҢмҠӨлҘј н—Ҳмҡ©н•  м¶ңмІҳмқҳ URL лҳҗлҠ” нҳёмҠӨнҠё IP мЈјмҶҢ

  • мҠӨнӮӨл§Ҳ(мҳҲ: http, https), нҸ¬нҠё лІҲнҳё, кІҪлЎң(path)лҠ” мғқлһөн•  мҲҳ мһҲлӢӨ.
    • origin = <scheme> "://" <host> [ ":" <port> ]
  • мҠӨнӮӨл§ҲлҘј мғқлһөн•ҳл©ҙ, л¬ём„ңмқҳ м¶ңмІҳ(origin)м—җм„ң мӮ¬мҡ©лҗң мҠӨнӮӨл§Ҳк°Җ мһҗлҸҷмңјлЎң м Ғмҡ©лҗңлӢӨ.
    • мҳҲлҘј л“Өм–ҙ, iframeм—җм„ң Content-Security-Policy: frame-ancestors *.example.comмңјлЎң м„Өм •н–Ҳкі  iframeмқҳ url мҠӨнӮӨл§Ҳк°Җ httpsлқјл©ҙ л¶ҖлӘЁм°ҪлҸ„ л°ҳл“ңмӢң https м—¬м•јн•ңлӢӨ.
  • CSP(Content-Security-Policy)м—җм„ң мҠӨнӮӨл§ҲлҘј м§Җм •н•ҳл©ҙ мӣҗм№ҷм ҒмңјлЎңлҠ” м •нҷ•н•ң мқјм№ҳк°Җ мҡ”кө¬лҗҳм§Җл§Ң, ліҙм•Ҳ лӢЁкі„к°Җ лҚ” лҶ’мқҖ мҠӨнӮӨл§ҲлЎң м—…к·ёл Ҳмқҙл“ңлҗң мҡ”мІӯмқҖ мһҗлҸҷмңјлЎң н—Ҳмҡ©лҗңлӢӨ.
    • http://example.comмқҖ https://example.comмқҳ лҰ¬мҶҢмҠӨлҸ„ н—Ҳмҡ©
    • ws://example.comмқҖ wss://example.orgмқҳ лҰ¬мҶҢмҠӨлҸ„ н—Ҳмҡ©
  • мҷҖмқјл“ңм№ҙл“ң(*)лҠ” м„ңлёҢлҸ„л©”мқё, нҳёмҠӨнҠё мЈјмҶҢ л°Ҹ нҸ¬нҠё лІҲнҳём—җм„ң мӮ¬мҡ©н•  мҲҳ мһҲмңјл©°, к°Ғ н•ӯлӘ©м—җм„ң к°ҖлҠҘн•ң лӘЁл“  мң нҡЁн•ң к°’мқҙ н—Ҳмҡ©лҗЁмқ„ мқҳлҜён•ңлӢӨ.
    • http://*.example.comмқҖ example.comмқҳ лӘЁл“  м„ңлёҢлҸ„л©”мқёмңјлЎңл¶Җн„° мҳӨлҠ” лҰ¬мҶҢмҠӨлҘј HTTP лҳҗлҠ” HTTPSлЎң н—Ҳмҡ©
  • кІҪлЎңк°Җ /лЎң лҒқлӮҳлҠ” кІҪмҡ°, к·ё кІҪлЎңлЎң мӢңмһ‘н•ҳлҠ” лӘЁл“  н•ҳмң„ кІҪлЎңлҘј л§Өм№ӯн•ңлӢӨ.
    • example.com/api/лҠ” лӢӨмқҢкіј к°ҷмқҖ кІҪлЎң лӘЁл‘җ н—Ҳмҡ©:
      • example.com/api/users
      • example.com/api/users/list
      • example.com/api/orders/123
  • л§Ңм•Ҫ кІҪлЎңк°Җ /лЎң лҒқлӮҳм§Җ м•ҠлҠ”лӢӨл©ҙ, м •нҷ•нһҲ к·ё кІҪлЎң н•ҳлӮҳл§Ң л§Өм№ӯн•ңлӢӨ.
    • example.com/api : example.com/apiл§Ң н—Ҳмҡ©

<scheme-source>

  • https:мҷҖ к°ҷмқҖ schemeмқ„ мӮ¬мҡ©н•  мҲҳ мһҲмңјл©°, мҪңлЎ (:)мқҙ н•„мҲҳм ҒмқҙлӢӨ
  • ліҙм•Ҳ м—…к·ёл Ҳмқҙл“ңк°Җ н—Ҳмҡ©лҗЁ
    • httpлҘј н—Ҳмҡ©н•ҳл©ҙ httpsлЎң лЎңл“ңлҗң лҰ¬мҶҢмҠӨлҸ„ н—Ҳмҡ©
    • wsлҘј н—Ҳмҡ©н•ҳл©ҙ wssлЎң лЎңл“ңлҗң лҰ¬мҶҢмҠӨлҸ„ н—Ҳмҡ©

upgrade-insecure-requests м§ҖмӢңм–ҙ

  • мӮ¬мқҙнҠёлҠ” л•Ңл•ҢлЎң мЈјмҡ” л¬ём„ңлҘј HTTPSлҘј нҶөн•ҙ м ңкіөн•ҳм§Җл§Ң лҰ¬мҶҢмҠӨлҠ” HTTPлҘј нҶөн•ҙ м ңкіөн•ңлӢӨ. <script src="http://example.org/my-cat.js"></script>

  • мқҙлҘј mixed contentлқјкі  н•ҳл©°, м•Ҳм „н•ҳм§Җ м•ҠмқҖ лҰ¬мҶҢмҠӨк°Җ мһҲмңјл©ҙ HTTPSм—җм„ң м ңкіөн•ҳлҠ” ліҙнҳёк°Җ нҒ¬кІҢ м•Ҫнҷ”лҗңлӢӨ.
    • лёҢлқјмҡ°м Җк°Җ кө¬нҳ„н•ҳлҠ” mixed content м•Ңкі лҰ¬мҰҳм—җ л”°лқј л¬ём„ңк°Җ HTTPSлҘј нҶөн•ҙ м ңкіөлҗҳлҠ” кІҪмҡ° м•Ҳм „н•ҳм§Җ м•ҠмқҖ лҰ¬мҶҢмҠӨлҠ” вҖңм—…к·ёл Ҳмқҙл“ң к°ҖлҠҘн•ң мҪҳн…җмё вҖқмҷҖ вҖңм°ЁлӢЁ к°ҖлҠҘн•ң мҪҳн…җмё вҖқлЎң л¶„лҘҳлҗңлӢӨ.
  • mixed contentм—җ лҢҖн•ң к¶Ғк·№м Ғмқё н•ҙкІ°мұ…мқҖ к°ңл°ңмһҗк°Җ лӘЁл“  лҰ¬мҶҢмҠӨлҘј HTTPSлҘј нҶөн•ҙ лЎңл“ңн•ҳлҠ” кІғмһ…лӢҲлӢӨ.
    • к·ёлҹ¬лӮҳ мӮ¬мқҙнҠёк°Җ лӘЁл“  мҪҳн…җмё лҘј HTTPSлҘј нҶөн•ҙ м ңкіөн•  мҲҳ мһҲлҚ”лқјлҸ„ к°ңл°ңмһҗк°Җ лҰ¬мҶҢмҠӨлҘј лЎңл“ңн•ҳлҠ” лҚ° мӮ¬мҡ©н•ҳлҠ” лӘЁл“  URLмқ„ лӢӨмӢң мһ‘м„ұн•ҳлҠ” кІғмқҖ м—¬м „нһҲ л§Өмҡ° м–ҙл ӨмӣҖ (м•„м№ҙмқҙлёҢлҗң мҪҳн…җмё мҷҖ кҙҖл Ёлҗң кІҪмҡ° мӮ¬мӢӨмғҒ л¶Ҳк°ҖлҠҘн•  мҲҳлҸ„ мһҲмқҢ).
  • upgrade-insecure-requests м§ҖмӢңл¬ёмқҖ мқҙ л¬ём ңлҘј н•ҙкІ°н•ҳкё° мң„н•ҙ мӮ¬мҡ©лҗңлӢӨ.
    • Content-Security-Policy: upgrade-insecure-requests
  • мқҙ м§ҖмӢңл¬ёмқҙ л¬ём„ңм—җ м„Өм •лҗң кІҪмҡ° лёҢлқјмҡ°м ҖлҠ” лӢӨмқҢкіј к°ҷмқҖ кІҪмҡ° лӘЁл“  HTTP URLмқ„ HTTPSлЎң мһҗлҸҷ м—…к·ёл Ҳмқҙл“ңн•ңлӢӨ.
    • лҰ¬мҶҢмҠӨ(мҳҲ: мқҙлҜём§Җ, мҠӨнҒ¬лҰҪнҠё л“ұ)лҘј лЎңл“ңн•ҳлҠ” мҡ”мІӯ
    • л¬ём„ңмҷҖ лҸҷмқјн•ң м¶ңмІҳмқё нғҗмғү мҡ”мІӯ(мҳҲ: л§ҒнҒ¬ лҢҖмғҒ)
    • мӨ‘мІ©лҗң лёҢлқјмҡ°м§• м»Ён…ҚмҠӨнҠёмқҳ нғҗмғү мҡ”мІӯ(мҳҲ: iframe)
    • form м ңм¶ң
  • н•ҳм§Җл§Ң, лҢҖмғҒмқҙ лӢӨлҘё м¶ңмІҳмқё мөңмғҒмң„ нғҗмғү(top-level navigation) мҡ”мІӯмқҖ м—…к·ёл Ҳмқҙл“ңлҗҳм§Җ м•ҠмҠөлӢҲлӢӨ.
    • Top-level navigationмқҖ лёҢлқјмҡ°м Җк°Җ нҳ„мһ¬ нҺҳмқҙм§Җм—җм„ң лӢӨлҘё нҺҳмқҙм§ҖлЎң м§Ғм ‘ мқҙлҸҷн•ҳлҠ” мҡ”мІӯмқ„ мқҳлҜё
      • <a href="..."> л§ҒнҒ¬лҘј нҒҙлҰӯн•ҙм„ң лӢӨлҘё нҺҳмқҙм§ҖлЎң мқҙлҸҷн•ҳкұ°лӮҳ,
      • мЈјмҶҢм°Ҫм—җ URLмқ„ м§Ғм ‘ мһ…л Ҙн•ҙм„ң мқҙлҸҷн•ҳкұ°лӮҳ,
      • лёҢлқјмҡ°м Җм—җм„ң нҺҳмқҙм§ҖлҘј мғҲлЎңкі м№Ён•ҳкұ°лӮҳ лӢӨлҘё мӣ№мӮ¬мқҙнҠёлЎң м§Ғм ‘ мқҙлҸҷн•ҳлҠ” л“ұ
  • мҳҲлҘј л“Өм–ҙ, https://example.orgмқҳ л¬ём„ңк°Җ upgrade-insecure-requests м§ҖмӢңм–ҙлҘј нҸ¬н•Ён•ҳлҠ” CSPмҷҖ н•Ёк»ҳ м ңкіөлҗҳкі  н•ҙлӢ№ л¬ём„ңм—җ лӢӨмқҢкіј к°ҷмқҖ л§ҲнҒ¬м—…мқҙ нҸ¬н•Ёлҗҳм–ҙ мһҲлӢӨкі  к°Җм •н–Ҳмқ„ л•Ң, лёҢлқјмҡ°м ҖлҠ” л‘җ мҡ”мІӯмқ„ лӘЁл‘җ мһҗлҸҷмңјлЎң HTTPSлЎң м—…к·ёл Ҳмқҙл“ңн•ңлӢӨ.
